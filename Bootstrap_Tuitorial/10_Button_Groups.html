Bootstrap Button Groups

Button Groups

Bootstrap allows you to group a series of buttons together (on a single line) in a button group by using a <div> element with class .btn group to create a button group

Vertical Button Groups

Use the class .btn-group-vertical to create a vertical button group

Justified Button Groups

To span entire width of the screen, use the .btn-group-justified class
